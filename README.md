# 7ewesbiya

**7ewesbiya** is a smart cultural guide application designed to enhance your travel experience. Explore cultural landmarks, get real-time recommendations, and enjoy interactive tours with our modern and intuitive mobile app.

## Features

-   **Interactive Cultural Tours**: Discover hidden gems and popular landmarks with guided tours.
-   **Location-based Recommendations**: Get suggestions based on your current location.
-   **Smart Audio Guides**: Listen to interesting stories and facts about the places you visit.
-   **Modern UI**: Experience a smooth and visually appealing interface with smooth animations.

## Getting Started

Follow these steps to get a copy of the project up and running on your local machine.

### Prerequisites

-   [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
-   An IDE (VS Code or Android Studio) with Flutter plugins.

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/yourusername/hewesbiya.git
    cd hewesbiya
    ```

2.  **Install dependencies:**

    ```bash
    flutter pub get
    ```

3.  **Run the app:**

    ```bash
    flutter run
    ```

## Tech Stack

-   **Framework**: [Flutter](https://flutter.dev/)
-   **Language**: [Dart](https://dart.dev/)
-   **State Management**: [Provider](https://pub.dev/packages/provider)
-   **Networking**: [Dio](https://pub.dev/packages/dio)
-   **Location**: [Geolocator](https://pub.dev/packages/geolocator)
-   **UI/Animations**: [Flutter Animate](https://pub.dev/packages/flutter_animate), [Avatar Glow](https://pub.dev/packages/avatar_glow)

## Folder Structure

The project structure is organized as follows:

-   `lib/core`: Contains core utilities, services (API, Location), and theme configurations.
-   `lib/features`: Contains the main feature modules (Home, Tour) with their respective screens and controllers.
